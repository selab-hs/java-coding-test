# 콜라 문제
* [링크](https://school.programmers.co.kr/learn/courses/30/lessons/132267)
* 10분

```java
class Solution {
    public int solution(int a, int b, int n) {
        int answer = 0;
        while(n >= a) {
            int cola =  (n / a)*b;
            int left = n % a;
            n = left + cola;
            answer += cola;
        }
        return answer;
    }
}
```
