# 기사단원의 무기
* [링크](https://school.programmers.co.kr/learn/courses/30/lessons/136798)
* 시간 초과로 답지 확인

```java
class Solution {
    public int solution(int number, int limit, int power) {
        int answer = 0;
        int[] num = new int[number];

        //약수 구하기
        for (int i = 0; i < number; i++) {
            for (int j = 1; j <= (int) Math.sqrt(i + 1); j++) {
                if ((i + 1) % j == 0) {  //i = 구해야하는 수 - 1
                    num[i]++;
                    if ((i + 1) / j != j) num[i]++;
                }
            }
        }

        //무게 구하기
        for (int n : num) {
            if (n > limit) answer += power;
            else answer += n;
        }
        return answer;
    }
}
```
number 가 100,000개에 들어가는 순간 시간 초과가 떠서 약수를 구하는 부분에서 루트를 사용하여 숫자 짝이 맞는 것을 이용하여 문제를 풀어냄

* 처음에는 for문으로 모두 순회하며 약수를 구하려고 했다 (시간 초과 날 줄 어느정도 예측하긴 하였다)
* 하지만 시간 초과를 보고서 동적 계획법을 사용하면 시간을 단축할 수 있을지 고민해서 풀다가 시간이 초과되어 답지를 확인하였다.
