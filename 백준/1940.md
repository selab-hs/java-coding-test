# 주몽의 명령
* 시간 : 10분
* 투 포인터를 사용하다가 while문 안에서 어떻게 처리해야할지 모르겠어서 답지 확인

# 코드
```java
import java.io.*;
import java.util.*;
class Main {
    public static void main(String[] args) throws IOException {
        BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
        StringTokenizer st = new StringTokenizer(br.readLine());
        int numberOfItems = Integer.parseInt(st.nextToken());
        st = new StringTokenizer(br.readLine());
        int sum = Integer.parseInt(st.nextToken());
        int[] value = new int[numberOfItems];
        st = new StringTokenizer(br.readLine());
        for (int i = 0; i < numberOfItems; i++) {
            value[i] = Integer.parseInt(st.nextToken());
        }
        int answer = 0;
        //값을 모두 받은 후 정렬
        int[] sorted = Arrays.stream(value).sorted().toArray();

        int start = 0;
        int end = numberOfItems - 1;

        while (start < end) {
            int cal = sorted[start] + sorted[end];

            if(cal < sum){
                start++;
            }else if(cal > sum){
                end--;
            }else {
                answer++;
                start++;
                end--;
            }
        }

        System.out.println(answer);
    }
}
```

# 배운 것
* 투 포인터를 사용하기 위해서는 정렬해야 함
* 더 많은 예제를 사용하면서 익숙해져야함
